# TimeScale Data Web API

Web-приложение для загрузки, обработки и анализа данных из CSV-файлов.

Проект разработан в рамках тестового задания для стажировки в ИнфоТеКС.

## Возможности

### Загрузка CSV

API принимает CSV-файл следующего формата:

```csv
Date;ExecutionTime;Value
2025-07-15T10:00:00.000Z;12.5;45.7
2025-07-15T10:00:15.000Z;10.2;43.1
```

После загрузки:

- выполняется полная валидация файла;
- данные сохраняются в PostgreSQL;
- рассчитываются агрегированные показатели;
- результаты сохраняются в отдельную таблицу.

Если файл с таким именем уже существует, данные полностью перезаписываются.

---

### Получение результатов

Поддерживаются фильтры:

- имя файла;
- диапазон времени запуска первой операции;
- диапазон среднего значения показателя;
- диапазон среднего времени выполнения.

---

### Получение последних значений

Возвращает последние 10 записей выбранного файла, отсортированные по времени начала выполнения.

---

## Валидация

При загрузке файла проверяются следующие условия:

- дата не раньше **01.01.2000**;
- дата не позже текущего момента;
- время выполнения не меньше **0**;
- значение показателя не меньше **0**;
- количество строк от **1** до **10 000**;
- все значения должны присутствовать и соответствовать своим типам.

Если хотя бы одно условие нарушено:

- данные не сохраняются;
- транзакция откатывается;
- пользователю возвращается соответствующая ошибка.

## Запуск проекта

### Через Docker

```bash
docker compose up --build
```

После запуска будут доступны:

| Сервис     | Адрес                         |
|------------|-------------------------------|
| Angular    | http://localhost:4200         |
| Swagger    | http://localhost:5062/swagger |
| PostgreSQL | http://localhost:5432         |

---

### Локальный запуск

Необходимо установить:

- .NET SDK 10
- Node.js
- PostgreSQL

Создать базу данных и указать строку подключения в `appsettings.json`.
